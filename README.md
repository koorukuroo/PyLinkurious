2. Start ipython(jupyter) and open **Basic_Example_Linkurious.ipynb**
3. Modified your linkurious ip at **linkurious_url**
  -  e.g., linkurious_url = 'http://127.0.0.1:8000'
4. Run all cell and you can meet Linkurious
